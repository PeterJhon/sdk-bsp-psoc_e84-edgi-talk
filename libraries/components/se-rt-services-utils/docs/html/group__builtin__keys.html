General Description

Built-In keys definitions.

SE RT Services provides the next built-in keys in two groups:

  • main built-in keys (from IFX_SE_KEY_ID_BUILTIN_HUK to IFX_SE_KEY_ID_BUILTIN_ATTEST_PUB)
  • test built-in keys (from IFX_SE_KEY_ID_BUILTIN_TEST_AES128 to IFX_SE_KEY_ID_BUILTIN_TEST_CMACKDF).

Main built-in keys used during boot and in the normal operating flow.

Test built-in keys used for test purpose (for application self-test as example).

All built-in keys have key location set to IFX_SE_KEY_LOCATION_SE.
