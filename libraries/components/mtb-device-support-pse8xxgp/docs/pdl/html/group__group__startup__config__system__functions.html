void SystemInit ( void  )

For CAT1D:

Initializes the system:

  • Calls the Cy_SystemInit() function.
  • Calls SystemCoreClockUpdate().

Gets core clock frequency and updates SystemCoreClock, cy_Hfclk0FreqHz, and cy_PeriClkFreqHz.
