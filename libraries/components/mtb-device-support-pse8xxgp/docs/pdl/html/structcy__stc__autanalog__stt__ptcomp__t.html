AUTONOMOUS ANALOG (Autonomous Analog Block) » PTComp (Programmable Threshold Comparator) » Data Structures

Description

The PTComp section in the State Transition Table state.

Used in cy_stc_autanalog_stt_t. For more details, refer to the device Architecture Technical Reference Manual.

Data Fields

bool unlock
 PTComp Unlock
FALSE - Locked:
Data from fields enableComp0, dynCfgIdxComp0 and enableComp1, dynCfgIdxComp1 will NOT be taken into account in the corresponding state of STT
TRUE - Unlocked:
Data from fields enableComp0, dynCfgIdxComp0 and enableComp1, dynCfgIdxComp1 will used in the corresponding state of STT.
